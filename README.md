# ship-Detection-From-SAR-using-GEE
This project leverages Sentinel-1 SAR imagery and Google Earth Engine to develop an automated ship detection workflow. It further includes a web-based application for visualizing detected ships and their movement tracks, enabling efficient monitoring of maritime activity.
